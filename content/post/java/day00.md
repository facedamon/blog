# 学习建议

- 记笔记，加注释，写总结
- 不要完全依赖于书籍和视频
- **建立有效的个人学习方法**
- 学习编程的捷径-->敲，狂敲
- 练习好打字，终极目标盲打
- 兴趣是学习编程最好的老师
